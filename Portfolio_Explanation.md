## 🎯 What This Project Is About

You have built **THREE impressive tools** that demonstrate Technical Art & VFX automation skills:

### 1. TechArt Toolkit (Unity + Unreal Engine)

A **cross-engine** toolkit with 5 modules for game development:

- **Shader Lab** - Create procedural textures using math (noise, SDF shapes)
- **VFX Performance Tester** - Compare optimized vs unoptimized effects
- **Lighting Tool** - Quick lighting setups for look development
- **Asset Optimizer** - Analyze meshes for performance
- **Procedural Generator** - Auto-create terrain and scatter objects

### 2. VFX Automation Suite (Python)

A **Python automation toolkit** for VFX artists to speed up repetitive tasks:

- Batch resize/convert images
- Convert image sequences to video
- Organize messy asset folders
- Quality check files before delivery

### 3. Spine Auto-Rig Tool (Python)

A **2D character rigging assistant** that:

- Takes a PSD or PNG of a character
- Uses **AI (MediaPipe)** to detect body pose automatically
- Maps layer names to bone names intelligently
- Generates a complete Spine project ready to animate

## 🤖 How You're Using AI

Your Spine Auto-Rig tool demonstrates **smart AI integration**:

```
Input: Character image → AI pose detection → Auto-bone mapping → Spine project
```

- **MediaPipe** (Google's AI) detects 33 body landmarks
- Your code translates those landmarks into body parts (head, torso, arms, legs)
- **Smart name mapping** understands multiple languages (English, Spanish)
  - Example: "brazo izq" → "upper_arm_L" (left arm bone)

This shows you understand how to:

- Integrate AI models into production pipelines
- Bridge AI output to artist-friendly tools
